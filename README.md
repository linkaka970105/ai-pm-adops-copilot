# AI 产品经理求职与面试复盘项目

本仓库用于沉淀 AI 产品经理求职准备资料，当前重点项目为 `AdOps Copilot`，包括产品方向确认、竞品分析、PRD 版本、后续简历与面试复盘材料。

## 目录结构

| 目录 | 内容 |
| --- | --- |
| `00_source_materials/` | 用户提供的原始资料 |
| `01_product_direction/` | 产品方向确认、业务流程、用户画像、痛点分析 |
| `02_competitive_analysis/` | 竞品分析、差异化策略 |
| `03_prd/` | PRD 草稿、拆分版本、需求说明 |
| `04_resume/` | 简历版本、项目经历、STAR 素材 |
| `05_interview_review/` | 面试转录稿、复盘结论、改进答案 |
| `06_question_bank/` | 面试题库、标准答案 |
| `07_portfolio/` | 作品集结构、项目故事线 |
| `08_outputs/` | 最终交付文档 |

## 维护约定

- 原始资料保留在 `00_source_materials/`，不覆盖、不删除。

- `.omx/` 等本地运行状态目录不上传到仓库。
